# Backend Boilerplate 
A Backend boilerplate to build a Node.js API, using:
  - Typescript
  - Express
  - Express-graphql
  - GraphQL Playground
  - Helmet
  - Cors
  - JWT Auth
  - Lodash/FP
  - Nodemon
  - Module-Alias
  - Jest
  - Moxios
  - Sinon
  - Cross-env
  - Prettier
  - ES-Lint
  - Production-ready setup (using PM2)

# Getting Started
1.	Clone the Repo
2.	run npm install
3.	create ou .env file by duplicating .env.template and inserting your environment values
4.	run npm start and navigate to <your base url>/playground

# Build and Test
Build command:

npm run build

Test:

npm test
